What is this?

A restaurant wine-pairing tool. Add your wine list, build food pairings, print guest cards and staff sheets, then record end-of-night ml usage for stock keeping.

Pair better. Move smarter. Track usage faster.

Built for venues that pair wine to food and need a practical bridge between the wine list, service decisions and POS stock keeping.

Service workflow

1. Add your wine list

Use Admin → Wines / import to add wines manually or upload the CSV template.

2. Build the menu pairing

Edit dishes, assign wines and set pour sizes.

3. Print service sheets

Guest card for the table. Staff sheet for notes, facts and open-drink pricing.

4. Save usage

Enter pairing sets and any individual glasses poured for stock tracking.
