# Wine Pairing App

Cloud-enabled restaurant wine pairing app.

Upload these files/folders to the root of the GitHub repository:

- `index.html`
- `vercel.json`
- `README.md`
- `supabase_schema.sql`
- `api/help.js`

Required Vercel environment variables:

- `SUPABASE_URL`
- `SUPABASE_ANON_KEY`
- `RESEND_API_KEY`

No `api/config.js` file is required in this version.
