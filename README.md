# Wine App V2 Demo

Static demo build for deploying as a second Vercel app while keeping the original app live.

## Deploy method
Create a new GitHub repository, upload these files, then import that repo as a new Vercel project.

Recommended Vercel project name: `wine-app-v2-demo` or `alsahwa-wine-app-v2`.

This is a standalone static demo. It does not replace the current live app unless you manually point the original domain to it.
